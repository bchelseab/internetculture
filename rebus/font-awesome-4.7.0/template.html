Larger Icons

This example uses embedded markup and css to display larger icons: 


  <div style="font-size:75px; color:DarkBlue;">
  <i class="fa fa-bicycle"></i>
  <i class="fa fa-transgender-alt"></i>
  </div>
  • This warning triangle is wrapped in a "span" tag with a class defining the font size.
